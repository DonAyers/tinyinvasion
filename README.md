# Tiny Invasion

Prototype scaffold for a Vite + React + TypeScript game shell with Phaser 3 and Zustand.

## Scripts

- `npm run dev` - start the Vite dev server
- `npm run build` - type-check and build for production
- `npm run lint` - run ESLint
- `npm run preview` - preview the production build

## Project structure

- `src/components` - React UI components, including the Phaser mount point
- `src/game` - Phaser bootstrap and scene files
- `src/state` - lightweight prototype state with Zustand
